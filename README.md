# 🚗 Austo Motor Company Customer Analysis

## 📌 Project Overview

This project focuses on Exploratory Data Analysis (EDA) of customer and automobile purchase data from Austo Motor Company. The objective was to analyze customer demographics, financial characteristics, and vehicle preferences to generate actionable business insights that could help improve marketing effectiveness.

---

## 🎯 Objectives

* Understand customer demographics and purchasing behavior
* Identify data quality issues and perform data cleaning
* Analyze individual variables using univariate analysis
* Explore relationships between variables using bivariate analysis
* Generate business recommendations based on findings

---

## 📊 Analysis Performed

### Data Cleaning

* Missing value identification and treatment
* Data consistency checks
* Correction of categorical value errors

### Univariate Analysis

* Customer demographics
* Income distributions
* Vehicle preferences
* Loan status analysis

### Bivariate Analysis

* Gender vs Vehicle Type
* Profession vs Vehicle Type
* Marital Status vs Vehicle Type
* Personal Loan vs Vehicle Type
* House Loan vs Vehicle Type

---

## 💡 Key Insights

* Sedan models showed the highest overall demand.
* Young adults formed the largest customer segment.
* Married customers preferred SUVs and Sedans.
* Single customers showed greater interest in Hatchbacks.
* Income and loan status influenced vehicle purchasing behavior.
* Customer demographics significantly impacted automobile preferences.

---

## 📈 Business Impact

The findings can help Austo Motor Company improve marketing campaigns, target customer segments more effectively, and optimize vehicle promotion strategies based on customer behavior patterns.
